# YEMC Website

Static website for Young Entrepreneurs Mugara Changemakers, a youth-led movement based in Mugara, Burundi.

## Files

- `index.html` - website content and sections
- `styles.css` - visual design and responsive layout
- `script.js` - mobile menu, current year, and newsletter confirmation

## Free Hosting Options

### GitHub Pages

1. Push this folder to a GitHub repository.
2. Open the repository on GitHub.
3. Go to Settings > Pages.
4. Under Branch, choose `main` and `/root`.
5. Save. GitHub will publish the site for free.

### Netlify

1. Go to https://www.netlify.com/.
2. Drag this project folder into Netlify Drop, or connect the GitHub repository.
3. Netlify will publish the site for free.

### Vercel

1. Go to https://vercel.com/.
2. Import the GitHub repository.
3. Keep the default static site settings and deploy.
